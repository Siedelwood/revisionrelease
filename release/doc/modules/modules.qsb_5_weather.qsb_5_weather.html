Module (5) Wetter

Dieses Modul ermöglicht das Ändern des Wetters.

Es können nun relativ einfach Wetterevents und Wetteranimationen kombiniert gestartet werden.

WeatherEvent

API.WeatherEventAbort ()
Bricht das aktuelle Event inklusive der Animation sofort ab.
API.WeatherEventCreate (_GFX, _Rain, _Snow, _Ice, _Monsoon, _Temp, _NotGrowing)
Erzeugt ein neues Wetterevent und gibt es zurück.

Ein Event alleine ändert noch nicht das Wetter! Hier wird ein Event definiert, welches an anderer Stelle benutzt werden kann. Das definierte Event kann jedoch in einer Variable gespeichert und immer wieder neu verwendet werden.

Hinweis: Es handelt sich um eine dynamische Wettersequenz. Dies muss beachtet werden! Eine statische Sequenz wird nicht funktionieren!

Parameters:

  • _GFX string Verwendetes Display Set
  • _Rain boolean Niederschlag aktivieren
  • _Snow boolean Niederschlag ist Schnee
  • _Ice boolean Wasser gefriert
  • _Monsoon boolean Blockendes Monsunwasser aktivieren
  • _Temp number Temperatur während des Events
  • _NotGrowing table Liste der nicht nachwachsenden Güter

Rückgabe:

    table Neues Wetterevent

Verwandte Themen:

  • API.WeatherEventRegister
  • API.WeatherEventRegisterLoop

Beispiel:

    -- Erzeugt ein Winterevent
MyEvent = API.WeatherEventCreate(
    "ne_winter_sequence.xml", false, true, true, false, -15,
    {Goods.G_Grain, Goods.G_RawFish, Goods.G_Honeycomb}
)
API.WeatherEventNext ()
Startet das nächste Wetterevent auf der "Wartebank". Wenn bereits ein Event aktiv ist, wird dieses gestoppt. Es erfolgt ein Übergang zum nächsten Event, sofern möglich.
API.WeatherEventPurge ()
Bricht das aktuelle Event ab und löscht alle eingereihten Events.

Mit dieser Funktion wird die komplette Warteschlange für Wettervents geleert. Dies betrifft sowohl einzelne Events als auch sich wiederholende Events.

API.WeatherEventRegister (_Event, _Duration)
Registiert ein Event für eine bestimmte Dauer. Das Event wird auf der "Wartebank" eingereiht.

Hinweis: Ein wartendes Event wird gestartet, sobald kein anderes Event mehr aktiv ist.

Parameters:

  • _Event table Event-Instanz
  • _Duration number Name des Events

Verwandte Themen:

  • API.WeatherEventNext
  • API.WeatherEventAbort
  • API.WeatherEventRegisterLoop

Beispiel:

    API.WeatherEventRegister(MyEvent, 300);
API.WeatherEventRegisterLoop (_Event)
Registiert ein Event als Endlosschleife. Das Event wird immer wieder neu starten, kurz bevor es eigentlich endet. Es darf keine anderen Events auf der "Wartebank" geben.

Parameters:

  • _Event table Event-Instanz

Verwandte Themen:

  • API.WeatherEventNext
  • API.WeatherEventAbort
  • API.WeatherEventRegister

Beispiel:

    API.WeatherEventRegister(MyEvent);
